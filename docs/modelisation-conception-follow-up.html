4. Package: Utilisateurs & Habilitations

Source: pj_mail_2.xlsx (2046 fonctions + 50 mappings)
Nouveauté v1.1: Profil OBSERVATEUR_ET_CIBLE pour les rôles doubles

4.1 ProfilFollowUp MIS À JOUR

Profil Description Rôles concernés
ADMINISTRATEUR Gestion complète du système Admins fonctionnels
OBSERVATEUR Peut saisir des observations uniquement Directeur Groupe, Directeur CEMM, Animateur
CIBLE_OBSERVATION Peut être observé uniquement CAC, Conseillers, Chargés d'affaires, Cash & Trade
OBSERVATEUR_ET_CIBLE Peut observer ET être observé DA, RCP, Directeur CAF, RCE
ANIMATEUR (= OBSERVATEUR) Supervision régionale (type: OBSERVATEUR) Animateurs (~23 codes ASAL) - Ne peuvent pas être observés
CONSULTATION Lecture seule Management, Reporting

4.2 Nouvelle entité: MappingFonctionRole

MappingFonctionRole NOUVEAU

Table de mapping entre les codes fonctions ASAL et les rôles Follow Up

  • id: UUID
  • codeFonctionASAL: String - Ex: "DIRAG", "CCPRO"
  • libelleFonctionASAL: String - Ex: "Directeur d'agence"
  • roleFollowUp: RoleFollowUp (FK)
  • profilFollowUp: ProfilFollowUp - OBSERVATEUR, CIBLE, ou OBSERVATEUR_ET_CIBLE
  • estDoubleRole: Boolean - True si peut observer ET être observé
  • actif: Boolean
  • remarque: String - Ex: "À identifier avec RH"

StatutPhaseDescription
