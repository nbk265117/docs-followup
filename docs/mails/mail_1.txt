Le système ORG permet principalement de gérer les entités, on y gère 3 niveaux : Région, groupe et agence  è Le référentiel Organisation repose sur Cinque grandes structure :

- Structure Etablissement : Représente les établissements (compagnies) à intégrer dans le référentiel Organisations. Par exemple : RMA ,BOA

- Structure Entité : Représente toutes les entités internes à l’établissement, par exemple : directions, départements, pôles...

- Structure point de vente : Représente les points de vente banque et assurance. Par exemple : Agent généraux, BGD, Courtiers, Agences Bancaires...

- Structure point de vente annexe : Représente les points de vente annexe à d’autres points de vente. Par exemple : les desk…

 

 

Les entités sont identifiées par des codes uniques è par des Codes unités Organisationnelles
La mise à jour du système ORG se fait à la demande è ou après nouvelle création d’entité ou un nouvelle nomination
Les régions sont composées de régions adjoints (sous-région/réseau) è les régions sont découpées en Région (N5) – Sous-région/Réseau (N4) – Groupe(N3) – Agence(N2) - Desk(N1)
