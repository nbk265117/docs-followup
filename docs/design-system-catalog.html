1 Button

<e-button>

Bouton avec plusieurs types, états, icônes et options de liste

Propriétés (Inputs)

Propriété Type Défaut Description
text string '' Texte affiché sur le bouton
type string 'primary' Type/style du bouton
loading boolean false Affiche un spinner de chargement
disabled boolean false Désactive le bouton
iconName string '' Nom de l'icône Material Icons
btnCta boolean false Style Call-to-Action (plus grand)
label string '' Label pour la liste de tags
list IItem[] [] Liste de tags à afficher sous le bouton
required boolean false Marque le champ comme requis
class string '' Classes CSS additionnelles
id string '' ID HTML du bouton

Types disponibles

primary secondary success danger warning info light dark link

Événements (Outputs)

Événement Type Description
(onRemove) any Émis lors de la suppression d'un tag de la liste

Exemples d'utilisation

<!-- Bouton simple --> <e-button text="Valider" type="primary"></e-button> <!-- Bouton avec icône --> <e-button text="Ajouter" type="success" iconName="add"></e-button> <!-- Bouton loading --> <e-button text="Chargement..." [loading]="true"></e-button> <!-- Bouton CTA --> <e-button text="Commencer" [btnCta]="true"></e-button> <!-- Bouton avec liste de tags --> <e-button text="Sélection" [list]="selectedItems" label="Éléments"></e-button>

2 Input

<e-input>

Champ de saisie avec support pour différents types, icônes, validation et mode téléphone

Propriétés (Inputs)

Propriété Type Défaut Description
label string '' Label du champ
type string 'text' Type HTML de l'input
typeInput string '' Type spécial: 'number' pour stepper
placeholder string '' Texte placeholder
value any '' Valeur du champ
required boolean false Champ obligatoire
error boolean false Affiche l'état d'erreur
errorMsg string '' Message d'erreur
disabled boolean false Désactive le champ
editable boolean true Rend le champ modifiable
iconName string '' Icône Material Icons
iconTitle string '' Titre de l'icône (tooltip)
unite string '' Unité affichée (ex: MAD, €)
min number - Valeur minimum (type number)
max number - Valeur maximum (type number)
step number 1 Pas d'incrémentation
phone boolean false Active le mode téléphone
options any[] [] Options pour le sélecteur (ex: devises)
control FormControl - FormControl Angular

Types HTML supportés

text email password number date datetime-local time tel

Événements (Outputs)

Événement Type Description
(onValueChanged) any Émis à chaque changement de valeur
(selectedValue) any Émis lors de la sélection d'une option

Exemples d'utilisation

<!-- Input simple --> <e-input label="Nom" placeholder="Entrez votre nom"></e-input> <!-- Input avec icône --> <e-input label="Recherche" iconName="search"></e-input> <!-- Input number avec stepper --> <e-input label="Quantité" type="number" typeInput="number" [min]="0" [max]="100" [step]="5"></e-input> <!-- Input avec erreur --> <e-input label="Email" [error]="true" errorMsg="Email invalide"></e-input> <!-- Input montant avec devise --> <e-input label="Montant" type="number" [options]="devises"></e-input> <!-- Input téléphone --> <e-input label="Téléphone" [phone]="true"></e-input>

3 Dropdown

<e-dropdown>

Liste déroulante simple avec recherche et sélection unique

Propriétés (Inputs)

Propriété Type Défaut Description
list any[] [] Liste des options (strings ou objets)
label string '' Label du dropdown
placeholder string '' Texte placeholder
value any '' Valeur sélectionnée
selectKey string '' Clé pour la valeur (objets)
showKey string '' Clé pour l'affichage (objets)
searchFlag boolean false Active la recherche
removeSearch boolean false Bouton pour effacer la recherche
showRemove boolean false Bouton pour effacer la sélection
required boolean false Champ obligatoire
error boolean false État d'erreur
errorMsg string '' Message d'erreur
disabled boolean false Désactive le dropdown
control FormControl - FormControl Angular

Événements (Outputs)

Événement Type Description
(selectedIndex) number Index de l'élément sélectionné
(selectedKey) any Clé de l'élément sélectionné
(valueSelected) string Valeur sélectionnée

Exemples d'utilisation

<!-- Dropdown simple (liste de strings) --> <e-dropdown [list]="['Option 1', 'Option 2', 'Option 3']" label="Choix" placeholder="Sélectionner..."></e-dropdown> <!-- Dropdown avec objets --> <e-dropdown [list]="villes" selectKey="id" showKey="nom" label="Ville"></e-dropdown> <!-- Dropdown avec recherche --> <e-dropdown [list]="pays" [searchFlag]="true" [removeSearch]="true" label="Pays"></e-dropdown>

8 Alert

<e-alert>

Alertes et toasts avec différents types et positions

Propriétés

9 Modal

<e-modal>

Fenêtre modale avec différentes tailles et types

Propriétés

title string Titre de la modal
titleDesciption string Description sous le titre
showModal boolean Affiche/masque la modal
modalWidth TypeSize Taille de la modal
type TypeMessage Type (affecte l'icône)
iconName string Icône personnalisée
hasIcon boolean Affiche l'icône
hasCloseButton boolean Affiche le bouton X
isSlideModal boolean Modal slide (depuis la droite)
staticBackdrop boolean Empêche fermeture par clic extérieur
actions IAction[] Boutons d'action

Tailles (TypeSize)

small medium large

Interface IAction

interface IAction { text: string; // Texte du bouton btnType: string; // Type du bouton (primary, secondary...) emit: string; // Valeur émise au clic btnWidth?: string; // Largeur du bouton }
<e-modal title="Confirmation" titleDesciption="Voulez-vous supprimer cet élément?" [showModal]="showDeleteModal" modalWidth="medium" type="warning" [hasIcon]="true" [hasCloseButton]="true" [actions]="[{text: 'Annuler', btnType: 'secondary', emit: 'cancel'}, {text: 'Supprimer', btnType: 'danger', emit: 'delete'}]" (actionEvent)="onModalAction($event)" > <p>Cette action est irréversible.</p> </e-modal>
Service disponible: ModalService permet d'ouvrir des modals programmatiquement.
